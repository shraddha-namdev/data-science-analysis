Trading Analysis (Fear vs Greed)
Objective

To analyze how market sentiment (Fear/Greed) affects trader performance and behavior.

Dataset
Bitcoin Market Sentiment Data
Hyperliquid Trader Data
Analysis
Compared PnL and win rate across sentiment
Analyzed trade frequency and position sizes
Studied long vs short behavior
Key Insights
Traders perform better during Greed periods
Trading activity increases during Greed
Fear periods show cautious trading behavior
Strategy
Reduce risk during Fear periods
Trade more actively during Greed but control risk
How to Run

Open the notebook (analysis.ipynb) and run all cells.

Tools Used
Python
Pandas
Matplotlib
